# ✨ Pro Multi-Agent Event Planner

An AI-powered dashboard that orchestrates multiple specialized agents to plan high-end events, generate visual mood boards, and optimize budgets in real-time.

**🌐 [Live Demo Link](https://multiagent-event-planner.streamlit.app/)** 

## 🚀 How it Works
This application uses a multi-agent architecture powered by **Groq (Llama 3.1)**:
- **Event Planner Agent:** Handles logistics, venues, and scheduling.
- **Creative Director Agent:** Designs a visual theme and selects a professional color palette.
- **Vendor Agent:** Researches and estimates market-realistic costs in AED.
- **Budget Optimizer Agent:** Refines costs to fit within your specified budget limit.

## 🛠️ Key Features
- **Dynamic Event Planning:** Tailors plans for weddings, corporate events, or workshops.
- **Visual Mood Board:** Provides Hex codes and design concepts.
- **Interactive Budget Table:** Edit "Optimized" prices directly in the app to see live savings calculations.
- **Data Export:** Download your final budget as a CSV file.
